== Description ==

Quran Shortcode helps you insert any Quranic text in your post easily by searching for a specific word or choosing the number of the verse and the number of the surah.

Features:

* Helps you insert any Quranic text in your post easily by searching for a specific word or choosing the number of the verse and the number of the surah.
* Using Shortcode.

PS: Owned by EDC [(E-Da`wah Committee)](https://edc.org.kw), For Support click to [Islam.com.kw](https://islam.com.kw).

== Installation ==

* Upload the Plugin to your blog, Activate it.
* In editor will be found Quran Shortcode button.
* Write any text from quran.
